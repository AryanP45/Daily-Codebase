# [6925. Faulty Keyboard](https://leetcode.com/problems/faulty-keyboard/)
> Version 1
```java
class Solution {
    public String finalString(String s) {
        int n = s.length();
        StringBuilder st = new StringBuilder();
        
        for(int i=0;i<n;i++)
           st = (s.charAt(i) != 'i') ? st.append(s.charAt(i)) : st.reverse();
            
        return st.toString();
    }
}
```
