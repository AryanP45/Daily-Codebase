# [2706. Buy Two Chocolates](https://leetcode.com/problems/buy-two-chocolates/)
> Version 1
```java
 class Solution {
    public int buyChoco(int[] prices, int money) {
        Arrays.sort(prices);
        return money -= (prices[0]+prices[1] <= money) ?
            prices[0]+prices[1] :
            0 ;

    }
}
```
