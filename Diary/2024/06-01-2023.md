# [485. Max Consecutive Ones](https://leetcode.com/problems/max-consecutive-ones/)
> Version 1
```java
 class Solution {
    public int findMaxConsecutiveOnes(int[] nums) {
        int max=0;
        int count=0;
        for(int i:nums){
            count += (i==1) ? 1 : -count;
            if(count>max) max = count;
        }
        return max;
    }
}
```
