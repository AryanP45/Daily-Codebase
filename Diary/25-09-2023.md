# [389. Find the Difference](https://leetcode.com/problems/find-the-difference/)
> Version 1
```java
 class Solution {
    public char findTheDifference(String s, String t) {
        int st=0,tt=0;
        int size=s.length();
        for(int i=0;i<size;i++){
            st += (int)s.charAt(i);
            tt += (int)t.charAt(i);
        }
        tt += (int)t.charAt(size);

        return (char)(tt-st);
    }
}
```
