# [2656. Maximum Sum With Exactly K Elements](https://leetcode.com/problems/maximum-sum-with-exactly-k-elements/)
> Version 1
```c++
 class Solution {
public:
    int maximizeSum(vector<int>& nums, int k) {
        sort(nums.begin(),nums.end());
        int answer=0;
        for(int i=0;i<k;i++){
            answer +=nums[nums.size()-1]++;
        }
        return answer;
    }
};
```
